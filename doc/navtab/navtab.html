创建navtab

创建一个navtab有以下两种方式：

  • Data属性：DOM添加属性data-toggle="navtab"后，单击触发。
    DOM示例1：　打开navtab
                            打开navtab

  • jQuery API：
    jQuery示例：　

    本例完整代码：

                            <script type="text/javascript">
                        function openMytab(obj) {
                            $(obj).navtab({id:'mynavtab', url:'doc/navtab/mytab.html', title:'我的业务页面'});
                        }
                        </script>

    jQuery API代码：

                            $(selector).navtab(options)
  • 取得当前navtab的内容容器：$.CurrentNavtab
  • 创建已存在相同ID的navtab时，如果url一致(未设置fresh参数时)，默认会直接切换到该navtab，否则覆盖已存在的navtab，可以增加reloadWarn参数以获得警告提醒
